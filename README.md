## Compatibility
Xcode 5.0+

## Installation
1. Copy the `.xccolortheme` file into the following directory:
`~/Library/Developer/Xcode/UserData/FontAndColorThemes`

2. Restart Xcode
3. Go to preferences and select **Solarized Dark** theme from the list
